近日书 5

· 2023-01-01 · # 近日书 # 发现
  • 《有顶天家族》：非常有趣的故事，只可惜最后居然犯了第一人称的视角错误。
  • 《有顶天家族·二代目归来》：依然有趣，依然在最后犯了人称视角错误。
  • 《太阳与少女》：从各处搜罗来的文字，算是扩展阅读么？
  • 《奔跑吧！梅洛斯：新解》：本没打算抱多大期待，却意外地好看。
  • 《热带》：嵌套的故事，很有趣的故事，结尾稍微弱了些。
  • 《糜骨之壤》：相比于奥尔加的前两本，这本就逊色许多了，一个魔怔的故事，一个魔怔的文本。
  • 《衣柜》：三个故事里衣柜还算不错，但总的来讲都只是以往文本重复变体，感觉不太用心。
  • 《怪诞故事集》：相比于之前的，更显得平庸了。
  • 《云游》：除了讲身体医学那部分，其他的都挺好的。
  • 《世界坟墓中的安娜·尹》：总觉得这样写是一种偷懒。
  • 《世界上最丑的女人》：总算水平又回来了，依稀又有了最开始读她的感觉，总的这么多本看下来，母题还是相对固定的。
  • 《成为乔布斯》：喧嚣散尽，方得以安静阅读。
  • 《故事: 材质·结构·风格和银幕剧作的原理》：我不怎么看电影，但里面很多东西对我还是深有启发，并且去看了卡萨布兰卡，确实不错，但如果是我的话，会设计成开头男主其实并没有拿到签证。
  • 《洛丽塔》：像部B级片。
  • 《独裁者手册 : 为什么坏行为几乎总是好政治》：失之于过于精巧的模型。
  • 《夜行》：森见写这类故事总感觉差一口气。
  • 《当我们在谈论爱情时我们在谈论什么》：冰山理论也好，留白也好，弦外之音也好，我真是太喜欢这种叙事形式了，但这种喜欢其实是更偏向海龟汤式的，和文学关系不大。
  • 《白鲸》：内核是很好的，但总觉得有太多基础性硬伤。
  • 《笛卡尔的错误》：因为缺乏神经医学的相关知识导致读着有些吃力，但确实帮助我再次拓展了知识边界，刷新了对心智的认识，引人深思。
  • 《剧变》：江郎才尽，不如《枪炮、病菌与钢铁》远甚。
  • 《战略级天使：不眠之梦》：比上一本差了一些，没那么精巧，却又稍显繁琐，但这种风格确实很好。
  • 《克苏鲁神话IV》：重感冒头晕之际正适合阅读。
  • 《我们》：是怎么与另外两本相提并论的？
  • 《万火归一》：十多年前友人送我的书，幻境重叠的感觉非常牛逼。
  • 《临高启明·第二卷》：熟悉的异界。
  • 《庞贝：一座罗马城市的生与死》：写得实在太好了，娓娓道来，学术性与文学性拉满。
  • 《大教堂》：这种故事方式实在太牛逼了，一切都刚刚好。
  • 《一朵桔梗花》：文笔实在太好，氛围实在太好，像是在看情绪片。
